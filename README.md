#Projeto ainda em finalização mas funcional

<h3> Etapas para rodar o APP </h3>

Inicie o backend

```bash
docker-compose up -d
```

Na pasta do frontnext instalar as dependências 

```bash
npm install
```

Inicie o servidor do frontend

```bash
npm run build
or
npm run dev
```

Acesse no navegador http://localhost:3000
